# Ace Editor

A minimalist code editor.

## Download
- [Firefox Extension](https://addons.mozilla.org/en-US/firefox/addon/ace-editor/)
- [Android App](https://addons.mozilla.org/en-US/android/addon/ace-editor/)

## Features
- Lightweight
- Fast performance
- Easy to use
