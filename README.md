# Skillrack Temporary Firefox Extension

A temporary Firefox extension made for **Skillrack** that enables:
- ✅ Copy
- ✅ Paste
- ✅ Edit Page functionality

This extension works **only on Firefox** and is loaded as a **temporary extension** for personal use.

---

## 📌 Features
- Enables copy and paste on Skillrack
- Allows editing web page only for you
- Lightweight and easy to load
- No permanent installation required

---

## 🦊 Supported Browser
- **Firefox only**
- Tested using Firefox’s *Temporary Extension* feature

---

## 📥 Installation Steps (Firefox)

1. Download the project **ZIP file**
2. **Extract / Unzip** the files
3. Open Firefox and go to:
   about:config
4. Click **“This Firefox”**
5. Click **“Load Temporary Add-on”**
6. Select the `manifest.json` file from the extracted folder
7. The extension will be installed and active immediately

> ⚠️ Note: Temporary extensions are removed when Firefox is restarted.  
You must reload it again if the browser is closed.
